## 📐 포스팅 작성 규칙
1. **파일명:** 영문 소문자 케밥케이스 (`ue5-nanite-tips.md`)
2. **제목:** 한국어 가능, 구체적으로
3. **태그:** 3~5개, 영문 + 한국어 혼용 가능
4. **코드 블록:** 언어 명시 필수 (`cpp`, `csharp`, `hlsl` 등)
5. **summary:** 검색 결과에 노출되는 한 줄 설명, 필수 작성
6. **draft:** 작성 중이면 `true`, 발행 준비 완료면 `false`

## 🚀 Hugo 명령어 치트시트
```bash
# 새 포스팅 생성 (archetypes 템플릿 자동 적용)
hugo new unreal/포스팅-제목.md
hugo new unity/포스팅-제목.md
hugo new project-logs/2026-04-05-프로젝트명.md

# 로컬 서버 실행 (draft 포함)
hugo server -D

# 빌드 (배포용)
hugo --minify

# 빌드 결과물 위치
# ./public/ 폴더 → GitHub Pages에 배포
```

## 🔌 MCP 연결 후 자동화 가능한 작업
- `"언리얼 Nanite 관련 포스팅 작성해줘"` → 파일 직접 생성
- `"오늘 작업 내용으로 프로젝트 로그 업데이트해줘"` → 파일 수정
- `"draft 상태 포스팅 목록 보여줘"` → 미발행 글 확인
- `"태그별 포스팅 현황 정리해줘"` → 콘텐츠 현황 파악
